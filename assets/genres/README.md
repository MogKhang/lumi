# Genre posters

Drop one 1000×1500 px JPG per genre here, named by slug:

- lowercase, spaces/symbols → `-`, `+` → `plus`
- examples: `action.jpg`, `science-fiction.jpg`, `martial-arts.jpg`,
  `mini-series.jpg`, `reality-tv.jpg`, `talk-show.jpg`, `tv-movie.jpg`,
  `18-plus.jpg`, `90s.jpg`, `lgbt.jpg`

The slug is generated by `genreSlug()` in `lib/utils/genre_assets.dart`.
A genre with a missing poster shows a placeholder card.
